#  Simon Game

An interactive **Simon Game** built using **HTML, CSS, and JavaScript**.  
This game tests your memory skills by challenging you to repeat an increasingly complex sequence of colors.

##  How to Play
1. Press any key to start the game
2. Watch the pattern of colors carefully
3. Repeat the sequence by clicking the buttons
4. Each level adds a new step to the sequence
5. If you make a mistake, the game ends 

---

##  Features
-  Random color sequence generation  
-  Increasing difficulty levels  
- Sound effects for each button  
-  Game over detection  
-  Restart functionality
